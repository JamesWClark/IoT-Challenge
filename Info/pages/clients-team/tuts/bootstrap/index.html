Step 1: Open your index.html file.

Step 2: Visit the Bootstrap website: http://getbootstrap.com/

Step 3: Browse through the Components section.

Step 4: Bootstrap provides a set of default styles. Visit cdnjs.com and get links to the latest version 3.
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/twitter-bootstrap/3.3.7/css/bootstrap.min.css">
Step 5: Bootstrap requires jQuery. Remember, typically JavaScript libraries go before the closing </body> tag.
<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.1.1/jquery.min.js"></script>
Step 6: And of course we need Bootstrap's JavaScript:
<script src="https://cdnjs.cloudflare.com/ajax/libs/twitter-bootstrap/3.3.7/js/bootstrap.min.js"></script>
Step 7: Remove the <h1>Site Under Construction</h1> message.

Step 10: Start thinking in grids. Bootstrap is built on a grid system of 12 units. For every row of content, you can add columns. Create a row and put a Jumbotron in it.
<div class="row">
  <div class="jumbotron">
    <h1>GrovePi Sensors</h1>
    <p>There be sensor data below, argh!</p>
    <p><a class="btn btn-primary btn-lg" href="#" role="button">Learn more</a></p>
  </div>
</div>
Step 11: Change the Jumbotron to identify and describe your project.
