Step 1: Create the following server.js file:
var express = require('express')
var app = express()

app.get('/', function (req, res) {
    res.send('Hello World!')
});

app.listen(3000, function () {
    console.log('Example app listening on port 3000!')
});
Step 2: Using command line, navigate to folder containing this file. For example, if your server.js file is located in Documents/NodeProject, you would change directories with the following command:
cd ~/Documents/NodeProject
Step 3: Now that you're in the right folder in your terminal, use the following command to run the web server:
node server
Step 4: Visit the page in your web browser: http://localhost:3000/

Step 5: Verify that you see Hello World! when you visit the above link.

Step 6: Before continuing on, notify your instructor when you've completed this task.
